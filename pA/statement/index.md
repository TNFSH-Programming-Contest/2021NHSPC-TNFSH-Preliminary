# 中位數

未完成，給定一個「遞增(減)數列」，多筆查詢區間中位數。 

\clearpage

## 輸入
第一行有 1 個正整數 $N$。  
第二行有 $N$ 個整數 $A_1, A_2, \cdots, A_N$。  
第三行有 1 個正整數 $Q$，表示接下來有 $Q$ 筆詢問。  
接下來 $Q$ 行，每行有 2 個正整數 $X$、$Y$，表示詢問第 $X$ 個至第 $Y$ 個之間（頭尾包含）的中位數為多少。  

## 輸出
對於每筆詢問，輸出該區間的中位數。  

## 輸入限制
 - $N \leq 10^6$
 - $1 \leq A_i \leq 10^9$
 - $Q \leq 10^6$
 - $Y - X$ 為2的倍數

## 子任務
\subtasks

\clearpage

## 範例輸入
```
5
1 2 3 4 5
2
1 5
3 5
```

## 範例輸出
```
3
4
```
